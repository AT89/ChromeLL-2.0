    ChromeLL 2.0 - the rewrite

    Project maintained by pokemans Hosted on GitHub Pages — Theme by mattgraham

    Install

    For normal use, install the extension from the Google Webstore

    Configure

    The extension will load with a default set of options, but you can always change them. Click "Options" on the right-click menu or on the popup to get to the options page.

    You can export your current configuration by clicking "Show ChromeLL Config" on the About page. To restore a config exported from either V1 or V2 of the extension, paste it in the box and press "Load ChromeLL Config"

    Features

    Helpful

    • HTML Tag buttons: bold, italic, underline, pre, spoiler, quote, mod
    • Batch uploader: ability to select multiple files, drag and drop image uploading
    • Number posts
    • Resize wide images

    Social

    • Highlight topics matching keywords - unlimited keywords, unlimited different colors
    • Highlight topics and posts by selected users
    • Notifications when a highlighted user posts

    A Better ETI

    • Remove topics and posts by selected users
    • Remove topics that match keywords:
      1. A comma separated, case-insensitive list, eg: autism, jeans
      2. A case-sensitive JSON array that uses javascript's match(), eg: ["NBA", "/^gosh,(.*)/i"]
